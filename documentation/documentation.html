Mr Goose
Responsive AJAX HTML Template

Getting Started

Introduction

Mr Goose is a professional responsive HTML5 & CSS3 template created for creative and portfolio websites.

If you like this plugin, feel free to rate it five stars     at your CodeCanyon account in Downloads section. If you encounter any problems, before rating the item, contact us so we can help you fix them.

How to use/install it

Unzip the file you downloaded and copy the content from folders sources/php or sources/html to your web server or where you need the website. I recomand using the PHP version (you will need to have PHP installed on your server) because it is split into components (header, footer, menu, ...).

Settings

Change config.js file

Go to assets/js/config.js and change the the variable's values.


Change or use your own background

To add your own background go to folder assets/gui/images/background-images/default and add 5 images in png format:
- left-background.png (image that wil be under the birds when content is on left)
- left-foreground.png (image that wil be over the birds when content is on left)
- light.png (image that can be a star ex moon or sun)
- right-background.png (image that wil be under the birds when content is on right)
- right-foreground.png (image that wil be over the birds when content is on right)
Got to assets/js/config.js and change backgroundImage variable value to default.

Note: Not all images are mandatory.

HTML Structure

Create a new page

Duplicate the files new-page.php (if you selected the PHP version) or new-page.html (if you selected the HTML version) and change the names.
Add in <div class="content"></div> node the page content.


Please add the title in the <h2></h2> node and the rest of the html code in <article></article> node.
